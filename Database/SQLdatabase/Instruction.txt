				Інструкція для створення бази даних
1.Заходите в MySQL Command Line

2.Вводите пароль який встановили при інсталяції  SQL

3.Вводите: 
				create database назва;
				use назва;

4.Cтворюєте таблички:
					source шляхДоФайлуУПапціTable
			наприклад:	source D:\Table\Role.sql
       
5.Додаєте зовнішні ключі(так як у пункті 4, лише файли з папки ForeignKey)

6.Додаєте значення в таблички(як і в попередніх, але папка  Insert)

	               Зауваження: 	Cпочатку IntoRole та IntoGrop, тоді IntoUser і нарешті IntoAssignment

P.S.	Якщо все вийшло, можете з'їсти цукерок :)
